Definition

In the case of RxJS, the main problem being solved is the ability to handle data that from asynchronous calls over multiple events. (examples: DOM Input events, Web Workers, Web Sockets)

RxJS combines the Observer pattern with the Iterator pattern and functional programming techniques used with collections to fill the need for an ideal way of managing sequences of events.

RxJS goes step further to represent all data with one abstraction. This is achieved by imagining all data as a stream.
